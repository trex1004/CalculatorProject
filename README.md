## 계산기 과제

* **사칙연산 구현**: `+`, `-`, `*`, `/` 연산자를 입력받아 안전하게 연산을 수행합니다.
* **연산 결과 기록 관리**: 내부 저장소(`List`)를 활용하여 연산된 결과를 누적하여 보관합니다.
* **예외 처리 (`try-catch`)**: 0으로 나누기(`ArithmeticException`) 및 잘못된 타입 입력(`InputMismatchException`) 발생 시 프로그램이 강제 종료되지 않고(`sc.nexLine`)으로 버퍼를 비워 안전하게 재입력을 유도합니다.
* **기록 관리 기능**: 가장 오래된 연산 기록을 삭제(`removeRecord`)하는 기능을 제공합니다.
